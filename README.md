# 🍔 Food Galaxy  

🚀 **Live Demo:** https://food-galaxy.vercel.app  

Food Galaxy is a modern **food ordering web application** built using React.js. It allows users to explore delicious food items, browse categories, and enjoy a clean, responsive UI for a seamless experience.

---

## 📌 Features  

- 🍕 Browse multiple food categories (Pizza, Burger, Sushi, etc.)  
- 🔍 Explore food items with images  
- 💰 View offers and pricing  
- 📱 Fully responsive design  
- ⚡ Fast and optimized performance  
- 🎯 Clean and user-friendly interface  

---

## 🛠️ Tech Stack  

- **Frontend:** React.js  
- **Styling:** CSS / Tailwind CSS  
- **Deployment:** Vercel  
- **State Management:** React Hooks
